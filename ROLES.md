# Navigator User Roles & Feature Access

This document outlines the different user roles in Navigator and the features available to each.

## User Roles

| Role | Description |
| :--- | :--- |
| **Free** | Standard user account with basic features and daily limits. |
| **Plus** | Premium tier with higher limits and access to automation tools. |
| **Pro** | Advanced premium tier with highest limits and full automation. |
| **Admin** | System administrator with full access to all features and debug tools. |
| **Tester** | Beta tester with early access to upcoming features. |

## Usage Limits

*   **Free Users**:
    *   3 job analyses per day.
    *   No automatic cover letter rewrites (manual refinement only).
    *   No email forwarding/automation.
*   **Plus Users**:
    *   Unlimited job analyses.
    *   Up to 5 automated job alerts per day via email forwarding.
    *   Up to 3 automatic cover letter refinement cycles by AI critic.
*   **Pro Users**:
    *   Unlimited job analyses.
    *   Up to 25 automated job alerts per day via email forwarding.
    *   Up to 3 automatic cover letter refinement cycles by AI critic.
    *   Access to high-reasoning AI models (Pro/Ultra) for complex analysis.
